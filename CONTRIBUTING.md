# Contributing

## Branching
- Base branch for development: `develop`
- Feature branches: `feature/SA-<id>-<short-name>`
- Bugfix branches: `bugfix/SA-<id>-<short-name>`
- Hotfix branches: `hotfix/SA-<id>-<short-name>`

## Workflow
1. Create branch from `develop`
2. Commit changes with clear messages
3. Open PR to `develop`
4. Pass CI checks
5. Get required approvals
6. Merge PR

## Commit style
Use Conventional Commits:
- `feat: ...`
- `fix: ...`
- `chore: ...`
- `docs: ...`
